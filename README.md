# IT Support Ticket Performance Dashboard Using Excel

## Project Overview
This project analyses IT support ticket performance using Microsoft Excel. The dashboard tracks ticket volume, ticket status, SLA breach rate, average resolution time, ticket priority, issue category, and assigned team performance.

The project uses dummy IT service desk data to simulate a real-world operational reporting scenario.

---

## Objectives
- Analyse IT support ticket volume
- Monitor open and closed tickets
- Calculate average resolution time
- Track SLA breach performance
- Identify common issue categories
- Analyse support team performance
- Build an interactive Excel dashboard for business reporting

---

## Dataset Overview
The dataset contains dummy IT support ticket records with fields such as:
- Ticket ID
- Created Date
- Resolved Date
- Priority
- Status
- Category
- Assigned Team
- Channel
- Customer Department
- SLA Hours

Additional calculated fields were created:
- Created Month
- Resolution Hours
- SLA Status
- Ticket Age Hours
- Priority Rank

## Key Insights
1. SLA breach rate is around 45.8%, showing that ticket resolution performance needs improvement.
2. There are 6 open tickets that require backlog monitoring.
3. Network and software issues appear frequently and should be reviewed for recurring problems.
4. Critical tickets require faster escalation due to strict SLA targets.
5. Team-level SLA performance should be monitored to identify workload or process issues.

---

## Business Recommendations
1. Implement daily SLA breach monitoring.
2. Escalate critical tickets earlier before they exceed SLA.
3. Review recurring network and software issues.
4. Track open ticket aging to reduce backlog.
5. Monitor SLA performance by assigned team on a weekly basis.
